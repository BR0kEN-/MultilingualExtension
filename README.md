## Installation

- `curl -sS https://getcomposer.org/installer | php`
- `vim composer.json`

```json
{
  "require": {
    kolev/multilingual-extension": "dev-master"
  },
  "config": {
    "bin-dir": "bin"
  }
}
```

- `composer install`
